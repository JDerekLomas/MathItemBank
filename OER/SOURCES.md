# OER Sources for K-12 Mathematics Item Banks

This document provides detailed information about sources for open educational mathematics resources, particularly assessment items and practice problems.

## Downloaded Resources

### 1. Khan Academy Exercises
- **Repository:** https://github.com/Khan/khan-exercises
- **Direct Link:** git clone https://github.com/Khan/khan-exercises.git
- **Last Updated:** Deprecated but stable (replaced by Perseus in 2015+)
- **Download Date:** November 1, 2025
- **File Count:** 453 HTML exercise files
- **License:** Creative Commons BY-NC-SA 3.0 (exercises), MIT (framework)
- **Format:** HTML/JavaScript interactive exercises
- **Grade Range:** K-12 and some college prep

### 2. OpenStax Exercises
- **Repository:** https://github.com/openstax/exercises
- **Direct Link:** git clone https://github.com/openstax/exercises.git
- **Active Development:** Yes
- **Download Date:** November 1, 2025
- **License:** AGPL-3.0
- **Format:** Ruby on Rails application with PostgreSQL database
- **Grade Range:** College level (some advanced high school)

## Additional Online Resources (Downloadable)

### 3. Mathematics Assessment Project (MAP)
- **Website:** https://www.map.mathshell.org
- **Organization:** Shell Centre, University of Nottingham
- **License:** CC BY-NC-ND 3.0

**Formative Assessment Lessons:**
- URL: https://www.map.mathshell.org/lessons.php
- Format: PDF + PowerPoint
- Count: 100 lessons
- Topics: Concept development and problem solving
- Download: Individual lesson downloads via download.php?fileid=[number]

**Summative Assessment Tasks:**
- URL: https://www.map.mathshell.org/tasks.php
- Format: PDF with rubrics
- Count: 94 tasks
- Categories: Novice, Apprentice, Expert
- Includes: Scoring rubrics and sample student work

**Professional Development:**
- URL: https://www.map.mathshell.org/pd.php
- Materials for teacher training in formative assessment

### 4. Illustrative Mathematics
- **Website:** https://illustrativemathematics.org
- **Free Access:** https://im.kendallhunt.com or https://www.geogebra.org/im
- **License:** CC BY 4.0 (most curriculum materials)
- **Grade Levels:** K-5, 6-8, 9-12
- **Format:** Online curriculum with downloadable PDFs
- **Note:** Pre/Mid/End-of-Unit Assessments not under CC license

**GitHub Presence:**
- Organization: https://github.com/illustrativemathematics
- Contains tools and utilities (not full curriculum)

### 5. Smarter Balanced Assessment Consortium (SBAC)
- **Sample Items:** https://sampleitems.smarterbalanced.org
- **Content Explorer:** https://contentexplorer.smarterbalanced.org
- **Grade Levels:** 3-8, High School
- **Subject:** Mathematics and ELA
- **Format:** Interactive online items
- **License:** Copyright Regents of UC (check terms for usage)
- **Count:** 104+ released items

### 6. PARCC Sample Items
- **Organization:** Partnership for Assessment of Readiness for College and Careers
- **Status:** Many states have transitioned away from PARCC
- **Note:** Check individual state education department websites for archived items

## Best Practices for Using OER

1. **Always check license terms** before redistributing or modifying
2. **Provide attribution** as required by license
3. **Keep copies of license files** with downloaded materials
4. **Document download dates** for versioning
5. **Review content quality** before use in instruction
6. **Contribute back** when possible to support the community

## Recommended Search Strategies

To find more OER math resources:

1. **GitHub:** Search for "math exercises", "mathematics assessment", "math questions"
2. **OER Commons:** Use advanced search with filters
3. **Creative Commons Search:** Filter by license type
4. **University Libraries:** Many maintain OER guides
5. **Professional Organizations:** NCTM, MAA often link to OER

---

## Quality Indicators for OER

Look for:
- ✓ Clear licensing information
- ✓ Regular updates or stable maintenance
- ✓ Alignment to standards (Common Core, state standards)
- ✓ Peer review or expert authorship
- ✓ Active community or organization support
- ✓ Accessibility features
- ✓ Multiple formats available
